## 💻 Installation

```bash
# Clone the laboratory
git clone https://github.com/I-Himanshu/my-script-laboratory.git

# Enter the lab
cd my-script-laboratory

# Make scripts executable
chmod +x ./setup/install.sh

# Run installation script
./setup/install.sh
```
